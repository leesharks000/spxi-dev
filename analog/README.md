# SPXI · Analog

**Ink-on-paper inscription as machine-eligible scholarly input.**

This section holds the SPXI Analog specification line — the extension of SPXI to non-digital inputs. Where the core SPXI protocol operates on digital entities inscribed with structured markup, SPXI Analog specifies how handwritten scholarly artifacts (marginalia, letters, notebooks, ink-and-paper drafts) enter the same protocol's semantic layer.

The bet: analog scholarly artifacts have always been machine-inaccessible because their evidentiary form (the physical inscription) does not survive the compression to digital tokens. SPXI Analog specifies the evidentiary form that does survive — the in-frame timestamped photograph — and the verification instrument that makes that evidentiary form usable at scale.
